

1. 기획안 (Cursor에 줄 요구사항)
1) 소개 (Introduction)

이 프로젝트는 브라우저에서 플레이 가능한 9×9 수도쿠 웹 게임을 React로 구현하는 것을 목표로 한다.
사용자는 마우스/키보드로 숫자를 입력하며, 규칙 위반(행/열/3×3 블록 중복)을 실시간으로 확인할 수 있고, 모든 칸을 올바르게 채우면 완료 메시지가 표시된다.

2) 기능 (Features)

9×9 그리드 UI

기본 수도쿠 보드 렌더링

고정 숫자(문제 제공 숫자)와 사용자가 입력하는 칸 구분 스타일

3×3 블록 구분선: 각 3×3 블록을 명확하게 구분하는 두꺼운 보더 스타일 적용

입력 & 검증

1~9 숫자만 입력 가능

행/열/3×3 블록 내 중복이 있으면 해당 칸을 에러 스타일로 표시

숫자를 지우면(빈칸) 에러 상태도 갱신

게임 상태 표시

에러가 없고, 모든 칸이 채워지면 "완료" 메시지 표시

오류 입력 횟수: 남은 빈칸 왼편에 규칙 위반 입력 횟수를 실시간으로 표시

남은 빈칸 수 표시

타이머: 게임 시작부터 종료까지 경과 시간을 mm:ss 형식으로 실시간 표시, 완료 시 자동 정지, 새 게임 시 리셋

숫자별 잔여 개수 표시: 보드 우측에 각 숫자(1-9)의 잔여 개수를 실시간으로 표시, 완성된 숫자는 초록색으로 하이라이트

숫자 완성 알림: 특정 숫자가 모두 채워지면 "Good! 숫자 X 완성" Toast 메시지가 화면 상단에 표시되고 2초 후 자동으로 사라짐

기본 컨트롤

새 게임(리셋) 버튼

힌트 버튼(선택: 정답 보드 존재 시 해당 칸 정답 넣기)

3) 시작하기 (Getting Started)

최소 요구사항

Node 18+

npm 또는 yarn

React SPA 형태로 동작

개발 모드는 npm run dev (Vite 기준) 또는 npm start (CRA 기준)

4) 설치 (Installation)
# Vite + React 예시
npm create vite@latest sudoku-react -- --template react
cd sudoku-react
npm install


Cursor에게:

src 폴더 안에 Sudoku 게임 코드(App.jsx, SudokuBoard.jsx 등) 를 생성하도록 요청

스타일은 src/index.css 또는 src/App.css 를 사용해서 작성

5) 사용법 (Usage)

로컬 실행

npm run dev    # Vite
# or
npm start      # CRA


브라우저에서 http://localhost:5173 (또는 CRA 포트) 접속 후

빈 칸 클릭 → 숫자 입력

규칙 위반 시 빨간색 하이라이트 및 오류 횟수 증가

모든 칸을 채우면 상단/하단에 완료 메시지 표시

오류 입력 횟수, 남은 빈칸, 타이머를 상단에서 실시간 확인

특정 숫자가 모두 채워지면 "Good! 숫자 X 완성" Toast 메시지가 화면 상단에 표시됨

보드 우측에서 각 숫자의 잔여 개수 확인 가능

3×3 블록이 두꺼운 선으로 명확하게 구분됨

6) 기술 스택 (Tech Stack)

Front-end : React, JavaScript

스타일링 : CSS (또는 Tailwind 사용 가능)

상태관리 : React Hooks (useState, useMemo)

빌드 도구 : Vite (또는 CRA)
